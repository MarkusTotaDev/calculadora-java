# Calculadora Java

Projeto de uma calculadora simples desenvolvido em Java para praticar conceitos de programação e versionamento com Git e GitHub.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão (com tratamento para divisão por zero)

## Tecnologias utilizadas
- Java
- Git
- GitHub

## Como executar
1. Clone este repositório
2. Abra o projeto em uma IDE Java (IntelliJ, Eclipse ou VS Code)
3. Execute a classe `Calculadora`

## Objetivo do projeto
Este projeto foi criado para praticar:
- Criação de repositório no GitHub
- Commits organizados
- Criação de branch
- Pull Request
- Merge de código
